
# Version History

## [v2.2.0-STABLE] - 2025-06-01
### Added
- **Final V2 Stabilization**: Comprehensive audit and cleanup of all core modules.
- **Dynamic Style Engine**: Enhanced task and card rendering to support any user-created grant categories gracefully.
- **Improved Analytics**: Recharts integration refined to handle multi-variate program data.
- **Identity Persona Overhaul**: Refined settings switcher with better visual cues for active roles.

### Fixed
- **Sync Protocol Consistency**: Standardized `BroadcastChannel` handlers to prevent message loopbacks.
- **UI Logic**: Fixed edge-case where chart tooltips would overlap modal overlays.
